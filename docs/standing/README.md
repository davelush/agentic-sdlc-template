# Standing

Material that doesn't flow through the shape → deliver → complete lifecycle and doesn't have a "state" in that sense: policies, legal, security, brand and marketing assets, and similar.

It lives in `docs/` because it's reference material the team needs, but it sits outside the lifecycle axis on purpose — don't try to force it into `future-roadmap`, `delivery`, or `current-reality`.

(Empty in this template — add subfolders like `policies/` as you need them.)
