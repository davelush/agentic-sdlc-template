# Future roadmap

Uncommitted thinking: what we *might* build. Roadmap items, ideas, the parking lot, and pitches that haven't been picked up. **Nothing here is a commitment.** When an item is picked up for delivery, it moves into [`../delivery/`](../delivery/) as a dated increment.

Keep speculative, long-horizon material light — it ages fast. The [parking lot](./parking-lot.md) is the low-ceremony place to drop an idea so it isn't lost.
