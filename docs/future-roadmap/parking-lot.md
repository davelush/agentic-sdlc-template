# Parking lot

Unprioritised ideas, captured so they aren't lost. No commitment, no order implied. Promote an item to a delivery increment by writing a pitch from [the template](../delivery/_templates/prd-pitch.md). (Example entries below — for the fictional Inkwell product.)

- **Email-on-publish** — notify subscribers when a post goes live. Natural follow-on to scheduled publishing.
- **Recurring schedules** — "every Tuesday at 9am" for serialised posts. Explicitly out of scope of the first scheduling increment; revisit if writers ask.
- **Best-time-to-publish suggestion** — recommend a send time from a writer's audience activity. Needs analytics we don't yet collect.
- **Calendar view** — see all scheduled posts on a month grid. Only worth it once writers routinely have several scheduled at once.
