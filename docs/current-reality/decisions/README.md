# Decision log (ADRs)

Architecture Decision Records — the settled choices that shape this codebase. An ADR is the reason a question should not be reopened without a superseding ADR. New ADRs are authored inside the delivery increment that makes the decision (from [the ADR template](../../delivery/_templates/adr.md)) and graduate here on increment close.
